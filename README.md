# braille-ocr
点字画像の検出実験用プロジェクトです。研究目的の途中成果を含みます。

## 概要
このリポジトリは、点字が写った画像から点の位置を検出する試験的な取り組みです。  
Blender を用いた合成データ生成と、YOLO を用いた物体検出の実験を中心に構成しています。

## 現状
### 合成データ
Blender により以下をランダム化した合成データを生成しています。

- 点の大きさ・形状
- カメラ角度
- 背景ノイズ  
  ※ 背景はRGBをランダムに配置させているため、少し不自然な色となっています(調整中)。

### 検出モデル
YOLO を用いて点字の6点をクラスとして学習しています。(Blenderで生成したランダムな6点の点字データを学習)
- Blenderのランダムな合成データは、一定の検出が可能

## 課題
- 実写点字は影や紙の凹凸が多様で、Blenderの合成データとの差が大きい
- 点の形状が不規則で、検出が安定しない
